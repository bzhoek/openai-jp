## Usage

```sh
open https://platform.openai.com/settings/profile?tab=api-keys
export OPENAI_API_KEY=sk-...
$ ./sentence.ts
Usage: sentence [options] [command]
A CLI for generating Japanese sentences using OpenAI's GPT-4o

Commands:
  everyday <word>              An everyday short sentence
  memorable <word>             A memorable short sentence
  speech [options] <sentence>  Save sentence as audio
  help [command]               display help for command

```
